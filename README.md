# Color Palette Generator

간단한 색상 팔레트 생성기입니다.

## 주요 기능

- 스페이스바를 눌러 새로운 색상 팔레트를 생성합니다.
- 색상을 잠그고 나머지 색상만 변경할 수 있습니다.
- 색상 코드를 클릭하여 클립보드에 복사합니다.
- 생성된 팔레트를 로컬 저장소에 저장하고 관리합니다.
- URL을 통해 팔레트를 다른 사람과 공유할 수 있습니다.

## 시작하기

```bash
git clone https://github.com/maro911220/color-palette-generator.git
cd color-palette-generator
```

```bash
npm install
```

```bash
npm run dev
```
